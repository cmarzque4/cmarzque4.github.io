Com funciona?

El nostre ninot combina elements visuals i electrònics per crear una experiència interactiva única:

  • 🔍 Exploració: Quan el nen toca una zona amb sensors, el ninot respon amb llums, sons o moviments.

  • 🤖 Intel·ligent: El sistema intern processa aquestes interaccions i adapta la resposta segons el programa configurat.

  • 🎯 Aprenentatge personalitzat: Els programes treballen habilitats com la coordinació motriu, l’atenció, el llenguatge i el reconeixement de colors i formes.

Com els ajuda?

  • Estimulació sensorial: Els colors, llums i sons desperten la curiositat i mantenen l’atenció.
  • Motricitat: Manipular les parts del ninot ajuda a millorar el control de moviments i la coordinació.
  • Aprenentatge actiu: Les respostes interactives del ninot reforcen el concepte de causa-efecte.
  • Confiança i expressió emocional: L'òs actua com a element amigable i segur, afavorint la comunicació i l’expressió de sentiments.
